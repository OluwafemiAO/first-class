How to make a web page with Flamingo
The only thing you have to give Flamingo is the "Content." Flamingo takes care of the rest, but you can take advantage of these options:
Custom URL
This is the part that goes after "flamingo.es/" like "flamingo.es/barry". Currently only accepts lowercase letters (a-z), numbers (0-9), and dashes (-). If you put anything else in there, flamingo will change it.
Custom Edit Code
Flamingo will give you a random edit code, but you can change it. You have to have this to make changes to the flamingo, and it can never EVER be retrieved if lost.
Title
Give your flamingo a relevant title so it shows up better when shared on social media, search results, and in browser windows.
Author
Let people know who made this! This shows up in some searches and social media posts, and your Twitter handle works really nicely if you post the link on Twitter.
Description
This is a short (200 character) summary of the page. This shows up in Twitter cards and search results. Flamingo will automatically use the first 200 characters of your flamingo as a description if you don't provide one.
Make your flamingo easy to read and understand
You can make your flamingo robust and full-featured with links, images, lists, headings, and more. Flamingo uses a popular set of rules called Markdown.
  • #Heading 1 = biggest heading
  • Heading 2 = second biggest heading
  • Heading 3 = third biggest
    • , and so on
  • Two spaces (space bars) at the end of a line starts a new line.
  • Return twice (leaving an empty line) starts a new paragraph.
  • *italics* = italics
  • **bold** = bold
  • Links:
    (http://flamingo.es) = link to flamingo (note: be sure to include the "http://" part of the link)
  • Images*: ![Monkey selfie](http://i.imgur.com/FXSBf8c.jpg)
  • Bulleted lists:
    1. - Bulleted item a
    2. - Bulleted item b
    3. - Bulleted item c
  • Numbered lists:
    1. Numbered item 1
    2. 2. Numbered item 2
    3. Numbered item 3
*Images must be uploaded somewhere else (we recommend imgur Because images can really slow down pages and we're all about "fast web pages for everybody, " images are not displayed by default. The reader has the option of displaying the images in the page or just viewing links to them. See an example at flamingo.es/images.
Important notes
  • Flamingo will delete any pages that are more than 6 months old but have been viewed less than two times ever. If you make a flamingo and forget about it, it might not be there 6 months later.
